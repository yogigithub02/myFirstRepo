SELECT * FROM employees

2. Write a query to get the department name and number of employees in the department.

SELECT in_department.department_name AS 'Department Name', COUNT(*) AS 'No of Employees' FROM in_department 
INNER JOIN emp_employee ON emp_employee.employee_department_id = in_department.department_id 
GROUP BY in_department.department_id, in_department.department_name 
ORDER BY in_department.department_name;
